# Умный AI поиск по Telegram-каналам (OpenRouter Edition)

Скрипт для автоматического сбора истории постов из любого публичного Telegram-каналов и их глубокого анализа с помощью бесплатных зарубежных и отечественных языковых моделей (LLM) через единый шлюз OpenRouter.

Инструмент умеет находить посты на заданную тему, выводить их текст прямо в консоль и генерировать **прямые кликабельные ссылки** на оригинальные публикации в Telegram.

## ✨ Особенности:
* **Полностью бесплатно:** Использует пул бесплатных моделей OpenRouter (Llama 3, Qwen, Mistral и др.).
* **Работает без VPN:** Запросы к API отправляются напрямую и стабильно работают из любой точки мира.
* **Удобная навигация:** Ссылки на найденные посты можно открывать кликом прямо из терминала.

1. **Клонируйте репозиторий:**
   ```bash

   cd tg-ai-smart-search

python -m venv .venv

source .venv/bin/activate

pip install requests beautifulsoup4 python-dotenv

Зарегистрируйтесь на сайте openrouter.ai.

Перейдите в раздел Keys -> Create Key.

Скопируйте созданный токен (начинается на sk-or-...).
4. **Настройте переменные окружения:**
Переименуйте файл .env.example в .env (или создайте его вручную в корне проекта).

Вставьте туда ваш ключ:
5. **Запустите поисковик:**
Введите ссылку на канал, интересующую тему и глубину просмотра, после чего ИИ выдаст результаты.
